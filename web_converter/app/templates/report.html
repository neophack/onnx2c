â±ï¸ MCU Timing & Measurement Advice

To accurately measure performance on ARM Cortex-M microcontrollers (like M3/M4/M7), it is recommended to use the hardware DWT (Data Watchpoint and Trace) cycle counter.

// 1. Enable DWT Cycle Counter
CoreDebug->DEMCR |= CoreDebug_DEMCR_TRCENA_Msk;
DWT->CYCCNT = 0;
DWT->CTRL |= DWT_CTRL_CYCCNTENA_Msk;

// 2. Measure
uint32_t start = DWT->CYCCNT;
model_inference(input, output);
uint32_t end = DWT->CYCCNT;
uint32_t cycles = end - start;

The estimated ROM and RAM shown above are based on static analysis of the generated model code (.o file). Actual usage may vary depending on your compiler settings (e.g., -Os vs -O3) and stack usage.
